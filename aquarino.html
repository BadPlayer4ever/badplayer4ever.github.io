AquarINO
Un controlador de acuarios Arduino

El proyecto AquarINO nace de la pasion por la acuariofilia
Combinando la tecnologia de tarjetas de desarrollo, nace la idea de automatizar y modernizar la forma de interactuar con nuestros acuarios plantados

Sensores y Actuadores

Un Arduino Uno R3 funciona de cerebro para controlar y coordinar los actuadores y sensores del AquarINO. dispositivos como Sensor de temperatura, Reloj RTC, Display OLED, Relays y sensores analogos dan funcionalidad al controlador.

Control del dispositivos

Capacidad de controlar hasta 3 dispositivos AC, ademas de tener su propio sensor de temperatura, y barra de luces RGB (No aptas para crecimiento de plantas)

Ideas para el futuro

De momento el controlador permite control manual de sus dispositivos, pero el objetivo del proyecto es avanzar hacia un modelo que integre encendido/apagado automatico de luces y alertas de temperatura como proyecto IoT.
Este proyecto no dispone de codigo fuente disponible al publico, dada su naturaleza de proyecto personal de investigacion.
